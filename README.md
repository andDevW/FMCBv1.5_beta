# FMCB v1.5 beta

FMCB v1.5 beta is an early pre-release version of FMCB that predates the FMCB splash screen.   
Compatible with PS2 launch consoles SCPH-30001 and SCPH-10000.  
Newer versions of FMCB support a wider range of PS2 models.  

Instructions are located in [Usage.txt](https://github.com/andDevW/FMCBv1.5_beta/blob/main/Usage.txt)
